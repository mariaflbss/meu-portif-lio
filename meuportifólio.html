API (Aprendizado por Projeto Integrado):


Objetivo: Desenvolver uma plataforma web interativa que permita a análise de desempenho dos municípios do Estado de São Paulo com base em dados de exportação e importação.


Resumo: O projeto consistiu na criação de um site com foco em visualizações gráficas que destacam informações sobre exportações e importações realizadas pelos municípios paulistas. A plataforma inclui filtros como valor agregado, valor FOB, peso líquido (kg), tipo de carga e município de origem/destino.

O objetivo é proporcionar uma ferramenta de análise comparativa para especialistas em logística, facilitando a tomada de decisões estratégicas com base em dados reais e visuais claros.


Minha Atuação: Atuei como desenvolvedora front-end, sendo responsável pela criação da página de feedback do usuário, pela responsividade geral do site e pela otimização dos gráficos de visualização no back-end.
Trabalhei com foco na experiência do usuário e na clareza dos dados exibidos, contribuindo diretamente para a usabilidade e funcionalidade da aplicação.


Tecnologias e Ferramentas Utilizadas:

GitHub - Figma - VS Code - HTML - CSS - Python - Flask - Google Colab - Jira - Git.
