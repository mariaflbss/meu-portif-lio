API - Aprendizado por Projeto Integrado

Objetivo: Desenvolver uma plataforma web interativa que permita a análise de desempenho dos municípios do Estado de São Paulo com base em dados de exportação e importação.

Resumo: Criamos um site com foco em visualizações gráficas, destacando informações de exportações e importações realizadas pelos municípios paulistas. A ferramenta possui filtros como valor agregado, valor FOB, peso líquido (kg), tipo de carga e município de origem/destino. O objetivo é fornecer aos especialistas em logística uma forma clara e interativa de analisar dados reais para tomada de decisões estratégicas.

Minha Atuação: Atuei como desenvolvedora front-end, sendo responsável pela criação da página de feedback do usuário, responsividade geral do site e otimização da exibição dos gráficos no back-end. Foquei fortemente na experiência do usuário, clareza dos dados e na funcionalidade visual da aplicação.

Tecnologias e Ferramentas:
GitHub – Figma – VS Code – HTML – CSS – Python – Flask – Google Colab – Jira – Git

Movin: Mobilidade Acessível para Todos

O Desafio: Mobilidade urbana é um direito, mas milhões de brasileiros com deficiências ainda enfrentam barreiras enormes ao utilizar o transporte público. A falta de sinalização acessível e informações claras torna a locomoção uma tarefa difícil e muitas vezes dependente de terceiros.

A Solução: Movin é um aplicativo inovador, desenvolvido para dar autonomia a pessoas com diferentes tipos de deficiência. Adaptado para deficiências visuais, auditivas e cognitivas, o app fornece informações em tempo real sobre trajetos e paradas de ônibus, com recursos específicos para cada necessidade:

  • Deficiência Visual: Alertas sonoros e narração do trajeto.
  • Deficiência Auditiva: Texto claro e ícones visuais com alto contraste.
  • Deficiência Cognitiva: Interface simplificada e linguagem acessível.

Impacto Esperado: Movin não é apenas um aplicativo, é uma ferramenta de inclusão social. Ele proporciona maior independência para pessoas com deficiência no transporte público, promovendo igualdade e segurança. Além disso, esperamos inspirar políticas públicas de acessibilidade, contribuindo para uma mobilidade urbana mais justa.

Inovação: Movin vai além da tecnologia, propondo uma mudança de mentalidade sobre mobilidade urbana. Criado dentro do Desafio Inova CPS, o projeto visa não só resolver um problema, mas transformar como entendemos a mobilidade acessível, garantindo que cada pessoa tenha liberdade e confiança para escolher o seu caminho.
